# Text Cleaning using NLTK

## Overview

This project cleans noisy text using Python and NLTK by removing punctuation, numbers, and stopwords.

---

##  Features

* Lowercasing
* Remove numbers & punctuation
* Tokenization
* Stopword removal

---

## Tech

* Python
* NLTK

---

## Run

```bash
pip install nltk
python day4.py
```

---

## Example

**Input:**
Wow!!! I bought this product for Rs.999 🤩

**Output:**
['wow', 'bought', 'product', 'rs']

## Outcome

Basic understanding of text preprocessing in NLP.
