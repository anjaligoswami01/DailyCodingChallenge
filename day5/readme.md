# 📰 News NLP Preprocessing Pipeline

A simple and modular **Natural Language Processing (NLP) pipeline** built using NLTK to preprocess news text data for machine learning applications.

---

## 🚀 Features

* Text lowercasing
* Sentence & word tokenization
* Stopword removal
* Punctuation cleaning
* Lemmatization
* Modular and reusable pipeline

---

## 📁 Project Structure

```
news-nlp-pipeline/
│── main.py
│── pipeline.py
│── preprocessing.py
│── data.py
│── requirements.txt
```

---

## 🛠 Tech Stack

* Python
* NLTK

---

## ▶️ How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the project:

```
python main.py
```
