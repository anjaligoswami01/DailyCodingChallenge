# 📄 Document Similarity Analyzer (NLP)

## Overview
This project demonstrates how to compute similarity between two text documents using **TF-IDF vectorization** and **cosine similarity**, a fundamental concept in Natural Language Processing (NLP).

## Key Concepts
- Text Vectorization (TF-IDF)
- Cosine Similarity
- Semantic Text Analysis

## Tech Stack
- Python
- Scikit-learn
- NLP Basics

## How It Works
1. Convert text into numerical vectors using TF-IDF
2. Compute cosine similarity between vectors
3. Output similarity score (0 to 1)
