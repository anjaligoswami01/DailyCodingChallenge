# NLP Task Day-03: Stemming vs Lemmatization

## 📌 Objective
To apply stemming and lemmatization techniques on product reviews using NLTK.

## 🔧 Tools Used
- Python
- NLTK

## 📊 Key Learnings
- Stemming removes suffixes but may produce incorrect words
- Lemmatization uses vocabulary and context for accurate results
- Text normalization improves NLP model performance

## 🚀 Output
Compared original, stemmed, and lemmatized text for multiple reviews.
